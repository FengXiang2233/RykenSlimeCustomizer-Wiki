# 材料生成器

```yaml
EXAMPLE_MATERIAL_GENERATOR:
  item_group: example_sub_group
  item:
    name: "&c示例材料生成器"
    material: IRON_BLOCK
  capacity: 10000
  output: [12]
  outputItem:
    material_type: full_slimefun
    material: EXAMPLE_ITEM
  tickRate: 5
  status: 4
  per: 10
```

### item\_group,item,recipe\_type,recipe,material\_type,material

详见[**物品**](broken-reference)

### capacity

生成一次所需电力。

### tickRate

生成间隔时间。

### status

设置显示信息的槽位。

### per

每次生成物品个数。
