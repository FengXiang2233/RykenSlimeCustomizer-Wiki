# 配方

设置合成物品、机器、geo资源等的配方配方格式跟SC大致相同，每一格都代表一个物品(**写法参考通用物品格式**)。

格式：

```
recipe:
  1: 
    ...
  2:
    ...
  3:
    ...
  4:
    ...
  5:
    ...
  6:
    ...
  7:
    ...
  8:
    ...
  9:
    ...
```

如果有些格子不需要放入物品，你可以这样写更快：

```
recipe:
  1:
    ...
  3:
    ...
  5:
    ...
  7:
    ...
  9:
    ...
```

当然如果这个物品（或机器或geo资源）是不可合成的，可以不用填recipe。
