# 主页

相信你或多或少的听说了一个自定义粘液附属SlimeCustomizer。但是它的局限性比较大。当然，这个插件(RykenSlimeCustomizer)也是一个自定义粘液附属，且比SlimeCustomizer有更多自定义性，如果你学过Java等这类语言，你就能很快上手写更高级的机器。
